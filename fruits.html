水果储藏指南

浆果类

储存要点

  • 温度：1-4℃
  • 湿度：90-95%
  • 建议位置：冰箱保鲜室
  • 储存时间：3-5天

储存方法

  1. 挑选：选择完整、无破损的果实
  2. 清理：不要提前清洗，食用时再清洗
  3. 包装：使用透气的保鲜盒
  4. 注意：避免挤压和堆叠

💡 小贴士：草莓等浆果类水果可以单层铺放，底部垫一张厨房纸，能更好地保持新鲜。

柑橘类

储存要点

  • 温度：4-8℃
  • 湿度：85-90%
  • 建议位置：冰箱保鲜室或阴凉处
  • 储存时间：1-2周

储存方法

  1. 挑选：选择适中成熟度的水果
  2. 清洁：用干布擦拭表面
  3. 包装：可以用网袋或保鲜袋松散包裹
  4. 环境：保持通风，避免潮湿

💡 小贴士：橙子、柚子等柑橘类水果不要与其他水果混放，以免互相影响。

热带水果

储存要点

  • 温度：12-15℃
  • 湿度：85-90%
  • 建议位置：室温阴凉处
  • 储存时间：视具体水果而定

储存方法

  1. 挑选：根据成熟度选择
  2. 清洁：保持表面干燥
  3. 摆放：避免阳光直射
  4. 检查：定期查看成熟度

💡 小贴士：香蕉、芒果等水果会释放乙烯，加速其他水果成熟，建议单独存放。

核果类

储存要点

  • 温度：0-4℃
  • 湿度：90-95%
  • 建议位置：冰箱保鲜室
  • 储存时间：3-7天

储存方法

  1. 挑选：选择适中硬度的水果
  2. 清洁：轻轻擦拭表面
  3. 包装：使用透气的保鲜袋
  4. 注意：避免碰撞和挤压

💡 小贴士：桃子、李子等核果类水果成熟较快，建议少量购买，及时食用。
